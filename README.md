# Lorena Achadinhos

Landing page responsiva para o grupo de promoções da Lorena Achadinhos.

## Publicar no GitHub Pages

1. Envie todos os arquivos desta pasta para a raiz do repositório.
2. No GitHub, abra **Settings → Pages**.
3. Em **Build and deployment**, escolha **Deploy from a branch**.
4. Selecione a branch `main` e a pasta `/root`.
5. Salve e aguarde a publicação.

O link do grupo do WhatsApp pode ser alterado no arquivo `index.html`, no atributo `href` do botão principal.
